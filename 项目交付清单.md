# 📦 春节模拟器 - 项目交付清单

## 🎯 项目概述

《春节模拟器》是一款生存模拟 + 叙事选择的Web小游戏，玩家需在9天春节假期中通过决策影响角色属性与剧情走向。

## 🎮 游戏特性

### 角色系统
- **10个独特角色**：每个角色有独特的人设、初始属性和故事线
- **6项核心属性**：存款、体重、面子、心情、健康、运气

### 事件系统
- **87个角色专属事件**：每个角色9个专属事件
- **15个常规事件**：可被多个角色触发
- **371个选项**：每个事件2-4个选项
- **蝴蝶效应**：选择影响后续剧情

### 结局系统
- **50个结局**：每个角色5个结局（2成功+1失败+1特殊+1隐藏）

### 技术特性
- **纯前端实现**：HTML5 + CSS3 + JavaScript
- **本地存档**：LocalStorage存储
- **响应式设计**：支持PC和移动端
- **GitHub Pages部署**：一键部署

## 🚀 快速开始

### 本地运行
```bash
# 进入src目录
cd src

# 用浏览器打开
open index.html
```

### GitHub Pages部署
1. 创建GitHub仓库
2. 上传src目录下的文件
3. 启用GitHub Pages
4. 访问 `https://yourusername.github.io/spring-festival-simulator/`

详细步骤见 `DEPLOY.md`
