# 《春节模拟器》常规事件 - 聚餐系列

> 本系列包含5个通用聚餐事件，适用于多个角色触发
> 融入热梗：预制菜占领年夜饭、79元眉笔、科技与狠活、锅气、感情深一口闷等

---

## 事件1：座位之争

**事件ID**: `dinner_seat_battle`

**事件名称**: 座位之争

**触发条件**: 
- 场景：年夜饭/家庭聚餐
- 触发时机：入座阶段
- 角色限制：无限制（所有角色均可触发）

**事件描述**:
```
年夜饭开席了！八仙桌旁，气氛突然变得微妙起来。

正对大门的主位空着——那是家族地位的象征。二舅公慢悠悠地走向那个位置，
却被大姑父抢先一步坐下："今年我买单，这个位置我坐定了！"

二舅公脸色一沉："按辈分，这个位置应该是我坐的。"

空气中弥漫着火药味，其他亲戚纷纷停下筷子，目光聚焦在这场"王位争夺战"上。

【系统提示】你坐在次席，正好处在风暴中心...
```

**选项与属性变化**:

| 选项 | 描述 | 属性变化 |
|------|------|----------|
| **A. 和稀泥大师** | "哎呀都是一家人，坐哪不是吃！来来来，咱们抓阄决定，公平公正！" | 情商+2，亲戚关系+1，威严-1 |
| **B. 站队二舅公** | "大姑父，辈分不能乱啊，二舅公是长辈，主位理应他坐。" | 威严+2，二舅公好感+2，大姑父好感-2 |
| **C. 站队大姑父** | "二舅公，大姑父今年确实出了大头，让他坐主位也是应该的嘛。" | 情商+1，大姑父好感+2，二舅公好感-2 |
| **D. 趁机上位** | "两位别争了，要不...这个位置让我坐？我给大家倒酒！" | 勇气+3，亲戚关系-2，获得"不知天高地厚"标签 |

**特殊效果**:
- 选择A：有50%概率成功化解矛盾，获得"和事佬"称号
- 选择D：有30%概率被亲戚群嘲，但获得"社交悍匪"隐藏成就

## 事件2：劝酒大战

**事件ID**: `dinner_wine_battle`

**事件名称**: 劝酒大战

**触发条件**:
- 场景：年夜饭/家庭聚餐
- 触发时机：酒过三巡
- 角色限制：年满18岁角色

**事件描述**:
```
酒过三巡，气氛渐入佳境。三叔端着酒杯，目光锁定了你。

"来来来，大侄子！感情深，一口闷；感情浅，舔一舔！
酒是粮食精，越喝越年轻！今天这杯酒你必须干了！"

你看着那满满一杯52度的白酒，喉咙发紧。

旁边的二姨也在起哄："就是就是，不喝就是不给我面子！
宁可胃上烂个洞，不叫感情裂条缝！"

【系统提示】你的酒量值正在经受考验...当前酒量：3/10 |
```

**选项与属性变化**:

| 选项 | 描述 | 属性变化 |
|------|------|----------|
| **A. 硬刚到底** | "三叔，感情铁，喝出血！这杯我干了，您随意！"（一饮而尽） | 勇气+3，酒量-2，亲戚关系+2，获得"酒神"临时buff（下回合骰子+1） |
| **B. 巧妙推辞** | "三叔，万水千山总是情，少喝一杯行不行？我明天还要开车呢！" | 情商+2，亲戚关系+1，威严-1 |
| **C. 甩锅转移** | "三叔，您先和我爸喝一个，我这边先敬一下二姨！"（趁机溜走） | 智商+2，三叔好感-1，二姨好感+1 |
| **D. 以茶代酒** | "三叔，只要感情有，喝啥都是酒！我以茶代酒，心意到了！" | 健康+1，三叔好感-2，获得"养生达人"标签 |

**特殊效果**:
- 选择A且酒量≤0：触发"醉酒"状态，下一事件强制跳过
- 选择B有50%概率被识破，需重新选择

## 事件3：预制菜风波

**事件ID**: `dinner_prepackaged_food`

**事件名称**: 预制菜风波

**触发条件**:
- 场景：年夜饭/家庭聚餐
- 触发时机：上菜阶段
- 角色限制：无限制

**事件描述**:
```
满桌菜肴陆续上桌，色香味俱全。你夹了一块"佛跳墙"，
口感确实不错，但总觉得少了点什么...

这时，眼尖的表哥突然拍案而起：
"等等！这佛跳墙...怎么跟我在XX直播间买的一模一样？
该不会是预制菜吧！"

全场寂静。

妈妈尴尬地笑了笑："这不是...这不是省事嘛...
现在预制菜可火了，79元就能买一大盒呢！"

爸爸脸色铁青："什么？我花了一下午'做饭'，
原来就是在加热料理包？"

【系统提示】检测到"科技与狠活"，家庭氛围正在波动... |
```

**选项与属性变化**:

| 选项 | 描述 | 属性变化 |
|------|------|----------|
| **A. 力挺预制菜** | "爸，哪里贵了？这么多年都是这个价格！预制菜解放劳动力，咱们有更多时间聊天不好吗？" | 智商+1，妈妈好感+2，爸爸好感-1，获得"李佳琦附体"梗 |
| **B. 支持爸爸** | "就是！年夜饭吃预制菜，一点锅气都没有！这年味都去哪了？" | 亲戚关系+1，爸爸好感+2，妈妈好感-1 |
| **C. 和稀泥** | "其实吧...预制菜也有好吃的，关键是看品牌。来，咱们先吃，边吃边聊！" | 情商+2，亲戚关系+1，威严-1 |
| **D. 抖机灵** | "至少比外卖强吧？68元的酸菜鱼，成本6.8元料理包——这才是真正的'科技与狠活'！" | 幽默+3，亲戚关系+1，获得"懂王"标签 |

**特殊效果**:
- 选择A：有30%概率触发爸爸"震怒"，下一回合强制进入"冷战"状态
- 选择D：有20%概率被妈妈瞪一眼，但全场爆笑

**事件ID**: `dinner_leftovers`

**事件名称**: 剩菜处理

**触发条件**:
- 场景：年夜饭/家庭聚餐
- 触发时机：用餐结束
- 角色限制：无限制

**事件描述**:
```
酒足饭饱，桌上却还剩下了大半桌菜。

妈妈看着满桌剩菜，叹了口气：
"这么多菜，扔了可惜，留着又吃不完..."

奶奶立刻接话："不能扔！年年有余懂不懂？
这些菜留着，明天、后天、大后天...够吃一个礼拜了！"

你看着那盘几乎没动过的"预制佛跳墙"，
心里默默计算：以这个分量，估计要吃到正月十五...

【系统提示】春节限定buff"剩菜循环"即将激活... |
```

**选项与属性变化**:

| 选项 | 描述 | 属性变化 |
|------|------|----------|
| **A. 打包带走** | "妈，这些剩菜我打包带走吧，正好省得做饭了！" | 节俭+2，妈妈好感+1，获得"剩菜终结者"称号 |
| **B. 支持奶奶** | "奶奶说得对！年年有余，这是好兆头！留着慢慢吃！" | 亲戚关系+2，奶奶好感+2，健康-1（暗示要吃好几天剩菜） |
| **C. 提议倒掉** | "妈，有些菜放久了不健康，该扔就扔吧，咱们不差这点。" | 健康+1，妈妈好感-1，奶奶好感-2，获得"不懂传统"标签 |
| **D. 创意改造** | "这些剩菜我可以做成'佛跳墙拌面'、'红烧肘子炒饭'，发小红书肯定火！" | 创意+3，亲戚关系+1，获得"美食博主"潜质 |

**特殊效果**:
- 选择B：下一回合开始时，有50%概率触发"剩菜吃腻了"负面状态
- 选择D：有10%概率真的火了，获得"意外网红"隐藏成就

## 设计说明

1. **普适性**：所有事件均为通用场景，不绑定特定角色
2. **趣味性**：融入2023-2024年度热门网络梗，增强共鸣
3. **策略性**：每个选项都有明确的属性权衡，没有绝对最优解
4. **延展性**：部分选项设置了概率触发的特殊效果，增加重玩价值
