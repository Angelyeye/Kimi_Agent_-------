## 🚀 部署指南

### GitHub Pages 一键部署

1. **创建GitHub仓库**
   - 登录GitHub，创建新仓库 `spring-festival-simulator`

2. **上传文件**
   ```bash
   cd /mnt/okcomputer/output/src
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/spring-festival-simulator.git
   git push -u origin main
   ```

3. **启用GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source 选择 `main` 分支
   - 点击 Save

4. **访问游戏**
   - 等待1-2分钟后访问 `https://yourusername.github.io/spring-festival-simulator/`

详细步骤见 `DEPLOY.md`

## 🎯 游戏特色

### 10个独特角色
1. **郝仕途** - 厅局风·编制内霸总
2. **花贝贝** - 都市隶人·回村Vivian
3. **范统** - 脆皮大学生·特种兵
4. **顾嘉** - 全职儿女·家务雇佣兵
5. **任兴** - 发疯学者·职场整顿版
6. **郝有乾** - 隐形富豪·土味房东
7. **毕成龙** - 鸡娃战神·绝望家长
8. **甄洋气** - 海归凡尔赛·假装名流
9. **胡三万** - 川渝雀圣·熬夜修仙党
10. **吴仁爱** - 大龄剩斗士·相亲遇难

### 核心系统
- ⏰ **时间系统**：9天 × 3时段（早/中/晚）
- 📊 **属性系统**：存款、体重、面子、心情、健康、运气
- 🎲 **事件系统**：专属事件 + 常规事件
- 🔗 **蝴蝶效应**：选择影响后续剧情
- 💾 **存档系统**：LocalStorage本地存储
