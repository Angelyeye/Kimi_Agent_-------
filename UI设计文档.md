# 春节模拟器 - UI设计文档

## 一、设计理念

### 1.1 设计愿景
《春节模拟器》的UI设计旨在创造一个**喜庆但不俗气、传统与现代融合**的游戏界面。我们希望玩家在进入游戏时就能感受到浓厚的春节氛围，同时享受到现代、流畅的交互体验。

### 1.2 设计关键词
- **喜庆** - 红色、金色主色调，春节元素点缀
- **现代** - 深色背景、玻璃拟态、流畅动画
- **沉浸** - 全屏界面、无缝切换、氛围装饰
- **易用** - 清晰的视觉层级、直观的操作反馈

### 1.3 参考案例
根据搜索结果，我们参考了以下设计趋势：
- **2024年UI设计趋势**：玻璃拟态(Glassmorphism)、新拟物化(Neumorphism)、暗黑模式
- **游戏UI设计**：深色主题+鲜艳强调色、卡片式布局、动态光影效果
- **春节主题设计**：红金配色、灯笼祥云元素、鎏金纹理

## 三、布局规范

### 3.1 响应式断点
| 断点 | 宽度 | 布局调整 |
|-----|------|---------|
| 移动端 | < 480px | 单列布局，角色3列，属性2列 |
| 平板 | 480px - 768px | 适配调整 |
| 桌面 | >= 768px | 完整布局，角色5列，属性6列 |

### 3.2 间距系统
| 名称 | 值 | 用途 |
|-----|---|------|
| xs | 4px | 微小间距 |
| sm | 8px | 元素内间距 |
| md | 16px | 组件间距 |
| lg | 24px | 区块间距 |
| xl | 32px | 大区块间距 |

### 3.3 圆角规范
| 名称 | 值 | 用途 |

| lg | 16px | 大卡片、面板 |

## 四、组件设计规范

### 4.1 按钮

#### 主按钮 (.btn-primary)
- 背景：红色渐变
- 文字：白色
- 阴影：红色发光
- 悬停：上移2px，增强阴影
- 点击：回到原位

#### 次要按钮 (.btn-secondary)
- 背景：半透明
- 边框：金色半透明
- 悬停：金色背景，金色边框

#### 发光按钮 (.btn-glow)
- 添加呼吸发光动画
- 用于开始游戏等重要操作

### 4.2 卡片

#### 角色卡片 (.character-card)
- 背景：深色半透明
- 边框：2px透明
- 选中状态：金色边框+阴影
- 悬停：上移+阴影

#### 属性卡片 (.stat-item)
- 背景：深色半透明
- 包含：图标、名称、数值、进度条
- 悬停：微缩放

### 4.3 进度条
- 高度：4px
- 背景：白色10%透明度
- 填充：根据属性不同颜色
- 动画：宽度变化有0.5s过渡

## 五、交互动效

### 5.1 页面切换
- 淡入动画：0.5s ease
- 元素依次进入：stagger效果

### 5.2 按钮交互
- 悬停：上移2px，0.3s过渡
- 点击：回到原位
- 发光按钮：呼吸动画2s循环

### 5.3 背景动画
- 灯笼：左右摇摆3s循环
- 祥云：漂浮8s循环
- 烟花：缩放淡出3s循环

### 5.4 属性变化
- 数值变化：缩放闪烁0.5s
- 进度条：宽度平滑过渡0.5s

### 5.5 事件切换
- 对话气泡：从下方淡入0.5s
- NPC头像：轻微弹跳2s循环
- 场景图标：漂浮3s循环

## 七、技术实现

### 7.1 技术栈
- **HTML5** - 语义化结构
- **CSS3** - 现代特性（Flexbox、Grid、动画、渐变）
- **JavaScript (ES6+)** - 游戏逻辑
- **LocalStorage** - 存档功能

### 7.2 CSS特性使用
- **CSS变量** - 统一的颜色、间距管理
- **Flexbox** - 弹性布局
- **Grid** - 角色卡片网格
- **backdrop-filter** - 毛玻璃效果
- **animation/transition** - 动画效果
- **gradient** - 渐变背景

### 7.3 响应式策略
- 移动优先设计
- 使用 `clamp()` 实现流体排版
- 媒体查询适配不同屏幕
- 触摸友好的按钮尺寸（最小44px）

## 九、设计原则总结

1. **一致性** - 使用统一的色彩、间距、圆角规范
2. **层次感** - 通过阴影、透明度、大小建立视觉层级
3. **反馈性** - 所有交互都有即时的视觉反馈
4. **沉浸感** - 全屏设计、氛围动画、无缝切换
5. **易用性** - 清晰的导航、直观的操作、响应式设计
