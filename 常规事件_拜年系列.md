# 《春节模拟器》常规事件 - 拜年系列

> 本系列包含5个通用常规事件，可被多个角色触发，融入春节传统习俗与现代网络热梗。

---

## 事件1: 红包大战

- **事件ID**: common_bainian_001
- **事件名称**: 红包大战
- **触发条件**: 拜访有小孩的亲戚家 或 自家有小孩来访（随机触发，概率25%）
- **事件描述**: 
  你刚坐下，亲戚家的小屁孩们就围了上来，眼睛直勾勾盯着你手里的包。"叔叔/阿姨过年好！"那声音甜得发腻。你摸了摸口袋里的红包，心里开始盘算：给多了肉疼，给少了没面子，不给？那是不可能的——大过年的！更头疼的是，对面家两个娃，你家只有一个，这账怎么算都是亏...

- **选项**:
  1. **【慷慨大方】每人塞888，图个吉利！** 
     - 存款-1776，面子+25，心情-10，亲戚好感+15
     - *"有钱任性！亲戚们纷纷夸你会来事，但你的钱包在哭泣..."

  2. **【精打细算】按人头给，一家600，公平合理**
     - 存款-1200，面子+10，心情+5
     - *"礼尚往来，谁也不欠谁。虽然没人夸你，但也没人挑理。"

  3. **【哭穷战术】"哎呀今年效益不好，意思意思哈"**
     - 存款-200，面子-15，心情+10
     - *"亲戚们表面笑嘻嘻，背地里已经开始八卦你'是不是混得不好'..."

  4. **【先发制人】不等他们开口，先夸对方孩子然后溜**
     - 存款-400，面子+5，心情+15，体力-5
     - *"哎呀这孩子长得真俊！叔叔阿姨还有事先走了哈！"——三十六计走为上！"

## 事件2: 被问工资

- **事件ID**: common_bainian_002
- **事件名称**: 灵魂拷问之工资篇
- **触发条件**: 与25岁以上亲戚独处超过3分钟（随机触发，概率30%）
- **事件描述**: 
  二舅端着茶杯慢悠悠地凑过来，脸上挂着"关心"的微笑："听说你在城里上班？一个月能拿多少啊？"还没等你回答，他又补了一刀："你表弟在县城当公务员，一个月五险一金加起来七八千呢，年底还有绩效..."你感觉空气突然凝固，这题怎么答都是送命题——说少了被同情，说多了被借钱，说不知道被认为在隐瞒...

- **选项**:
  1. **【实话实说】"也就一万出头，勉强够花"**
     - 存款+0，面子+5，心情-10，亲戚好感+5
     - *"二舅点点头：'还行吧，但还是要节约，现在房价贵...'然后开始了长达半小时的人生教诲。"

  2. **【低调保命】"没多少，够吃饭就行"**
     - 存款+0，面子-5，心情+5
     - *"二舅意味深长地看了你一眼：'年轻人要上进啊，不能安于现状...'"

  3. **【反客为主】"二舅您退休金多少？我帮您算算够不够养老"**
     - 存款+0，面子+10，心情+15，亲戚好感-10
     - *"二舅脸色一变，借口去厕所溜了。你成功化解危机，但可能被拉入'不懂礼貌'黑名单..."

  4. **【糊弄学大师】"嗨，也就那样，比上不足比下有余"**
     - 存款+0，面子+0，心情+10
     - *"说了等于没说，但谁也没法挑毛病。糊弄学，永远的神！"

## 事件3: 被问对象

- **事件ID**: common_bainian_003
- **事件名称**: 催婚大作战
- **触发条件**: 角色年龄≥22岁且未婚（随机触发，概率35%）
- **事件描述**: 
  七大姑八大姨不知何时已经把你团团围住，那架势堪比三堂会审。"有对象了吗？""什么时候带回来？""要不要阿姨给你介绍一个？"三连击打得你措手不及。大姨更是祭出了杀手锏："你看隔壁老王家的闺女，比你小两岁，孩子都会打酱油了！你再不抓紧，好人都被挑光了！"你感觉头顶有三只乌鸦飞过，这熟悉的配方，这熟悉的味道...

- **选项**:
  1. **【卑微求饶】"在找了在找了，明年一定带回来！"**
     - 面子-10，心情-15，亲戚好感+10
     - *"你立下了明年必还的flag，亲戚们满意地散去。但明年的你该怎么办？"

  2. **【硬核回怼】"我不找对象主要是因为您"**
     - 面子+15，心情+20，亲戚好感-20
     - * *"亲戚：'跟我有啥关系？' 你：'对呀，我找不找对象跟您有啥关系？' ——绝杀！"

  3. **【转移话题】"哎呀大姨，您这毛衣在哪买的？真好看！"**
     - 面子+5，心情+10
     - *"大姨果然上当，开始滔滔不绝讲她的购物经。催婚危机暂时解除！"

  4. **【卖惨战术】"唉，我喜欢的不喜欢我，可能这辈子就这样了"**
     - 面子-5，心情-5，亲戚好感+5
     - *"亲戚们反而来安慰你：'没事没事，缘分未到，别着急...' 以退为进，高！"

## 事件4: 熊孩子拆家

- **事件ID**: common_bainian_004
- **事件名称**: 熊孩子入侵
- **触发条件**: 家中有6岁以下儿童来访（随机触发，概率20%）
- **事件描述**: 
  "他还是个孩子！"这句话还没落下，你珍藏多年的手办已经身首异处。亲戚家的小霸王在你房间里横冲直撞，你的乐高城堡变成了废墟，游戏手柄被当成了磨牙棒，墙上多了几幅"抽象派涂鸦"。孩子他妈在旁边笑呵呵地说："大过年的，别跟孩子计较嘛~"你看着满地的狼藉，血压直线飙升，但"大过年的"四个字像紧箍咒一样把你钉在原地...

- **选项**:
  1. **【忍气吞声】"大过年的...算了算了..."**
     - 心情-20，面子+5，物品损失（随机）
     - *"你默默收拾残局，心里默念：大过年的、大过年的、大过年的..."

  2. **【温柔劝阻】"小朋友，这个不能玩哦，叔叔/阿姨给你糖吃"**
     - 存款-50，心情-5，面子+10
     - *"熊孩子接过糖，暂时消停了。但你发现糖纸被塞进了你的键盘缝隙..."

  3. **【以牙还牙】"哎呀，小朋友这么活泼，来让叔叔/阿姨抱抱！"（疯狂rua头）**
     - 心情+10，面子+5，亲戚好感-5
     - *"你把熊孩子的发型rua成了鸡窝，他哭着找妈妈去了。小胜一局！"

  4. **【机智转移】"走！叔叔/阿姨带你去楼下放炮仗！"**
     - 体力-10，心情+15，存款-30（买鞭炮）
     - *"成功把祸害转移到了户外！虽然累个半死，但至少房间保住了。"

## 事件5: 长辈说教

- **事件ID**: common_bainian_005
- **事件名称**: 人生导师时间
- **触发条件**: 与60岁以上长辈独处（随机触发，概率40%）
- **事件描述**: 
  爷爷/奶奶/外公/外婆拉着你的手，眼神里满是慈爱（和话痨的前兆）。"孩子啊，爷爷/奶奶跟你说..."接下来的一个小时，你听到了：他们年轻时如何艰苦奋斗、现在的年轻人如何吃不了苦、你应该如何规划人生、那个谁谁谁（别人家的孩子）现在混得多好..."都是为你好"这句话出现了不下十次。你坐得屁股发麻，但看着长辈慈祥的面容，又不忍心打断...

- **选项**:
  1. **【乖乖听讲】"嗯嗯，您说得对，我记住了"**
     - 心情-10，体力-10，长辈好感+20
     - *"你成了最懂事的孩子，但你的灵魂已经出窍..."

  2. **【虚心请教】"那您说我该怎么办？"**
     - 心情-15，体力-15，长辈好感+25
     - *"恭喜你，成功开启了无限套娃模式。长辈更来劲了，说教时间翻倍！"

  3. **【阳奉阴违】表面点头，心里盘算晚上吃什么**
     - 心情+5，体力-5，长辈好感+10
     - *"左耳进右耳出，人生必备技能。你甚至学会了在适当的时候点头和叹气。"

  4. **【借故开溜】"哎呀，我妈叫我帮忙包饺子！"**
     - 心情+15，体力-5，长辈好感+0
     - *"成功脱身！虽然被妈妈瞪了一眼（她根本没叫你），但总比听说教强。"

## 设计说明

### 融入的热梗清单

| 热梗 | 出处/含义 | 应用事件 |
|------|----------|----------|
| 大过年的 | 中国八大原谅之一 | 全部事件 |
| 都是为你好 | 长辈经典语录 | 长辈说教 |
| 别人家的孩子 | 童年阴影系列 | 被问工资、长辈说教 |
| 他还是个孩子 | 八大原谅之一 | 熊孩子拆家 |
| 00后整顿亲戚圈 | 网络流行语 | 被问对象 |
| 糊弄学 | 社交生存技巧 | 被问工资 |
| 来都来了 | 八大原谅之首 | 红包大战 |
| 给个面子 | 社交绑架用语 | 红包大战 |
| 夺命三连问 | 亲戚问候套餐 | 被问对象、被问工资 |

### 属性制衡设计

- **存款 vs 面子**: 红包事件中，花钱换面子或省钱丢面子
- **心情 vs 亲戚好感**: 怼亲戚爽了自己但得罪人，忍气吞声委屈自己但和谐
- **体力 vs 心情**: 溜号需要体力但能保心情，硬扛消耗心情但省体力

### 事件触发概率

- 高概率（35-40%）: 被问对象、长辈说教（春节高频场景）
- 中概率（25-30%）: 被问工资、红包大战
- 低概率（20%）: 熊孩子拆家（需要特定条件）
